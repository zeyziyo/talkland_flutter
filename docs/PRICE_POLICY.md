# Monetization Strategy (수익화 전략) - Talkie

이 문서는 Talkie 앱의 지속 가능한 서비스를 위한 수익화 전략을 정의합니다.
**핵심 철학:** 광고 수익을 통해 서버 비용을 충당하고, 사용자에게는 부담 없는 학습 환경을 제공하는 **Zero-Risk 모델**입니다.

## 1. 기본 정책 (Free Tier)
모든 사용자는 기본적으로 무료로 서비스를 이용할 수 있습니다.

*   **기본 번역 제공:** 매일 **5회** 무료 번역 제공
*   **회화 연습(STT/TTS):** **무제한** (기기 내장 엔진 사용으로 비용 0원)
*   **복습(Review):** **무제한** (저장된 오프라인 데이터 활용)

## 2. 수익화 모델 (Business Model)

### A. 보상형 광고 (Rewarded Ads) - ⭐️ 핵심 수익원
사용자가 기본 제공 횟수(5회)를 모두 소진했을 때 작동합니다.
*   **매커니즘:** 30초 동영상 광고 시청 시 -> **번역 5회 추가 충전**
*   **수익성:** 광고 1회 수익(약 13원) > API 5회 사용 비용(약 0.65원) = **약 20배 마진**
*   **특징:** 사용자가 공부를 많이 할수록 개발자의 수익도 함께 증가하는 선순환 구조.

### B. 광고 제거 (Ad-Free Pro) - 선택적 유료화
광고 시청을 원하지 않는 사용자를 위한 **구독형** 상품입니다.
*   **가격:** 월 **₩2,900** (커피 한 잔 값)
*   **혜택:**
    *   **월 1,500회** 번역 (일 평균 50회, 광고 시청 없이 즉시 번역)
    *   앱 내 모든 배너/전면 광고 제거
    *   프리미엄 학습 자료 접근 권한 (향후 추가)

## 3. 리스크 관리 (Risk Management)
*   **비용 방어:** 무료 사용자(일 5회) 및 유료 사용자(월 1,500회) 모두에게 합리적인 사용 한도를 적용하여 고정비 리스크를 제거합니다.
*   **안정성:** Google Cloud Translation API(정식 유료)를 사용하여 서비스 중단 없이 안정적인 품질을 보장합니다.

---
**요약:**
"기본 5회 + 광고 보고 충전" 모델을 통해 **사용자 100만 명이 되어도 적자 걱정 없는** 탄탄한 수익 구조를 구축합니다.
