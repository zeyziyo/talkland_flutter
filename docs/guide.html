Advanced: Create Your Own Study Materials

You can import your own list of words and sentences using a JSON file. This is perfect for studying specific topics like "Travel", "Business", or "TOEIC".

1. JSON File Format

Create a text file with the extension .json (e.g., travel_words.json). Copy the format below:

{
  "title": "Essential Travel Phrases",
  "default_type": "sentence",
  "translations": [
    {
      "source": "Where is the bathroom?",
      "target": "ÌôîÏû•Ïã§Ïù¥ Ïñ¥ÎîîÏóêÏöî?",
      "context": "Restaurant"
    },
    {
      "source": "Delicious",
      "target": "ÎßõÏûàÏñ¥Ïöî",
      "type": "word",
      "context": "Food"
    }
  ]
}
  • title: The name of your study set.
  • default_type: "word" or "sentence" (applies to all items unless overridden).
  • context: (Optional) Adds a helpful tag like "Airport" or "Greeting".

2. Generate with AI (ChatGPT / Copilot)

Don't want to type it manually? Ask AI to do it for you! Copy and paste this prompt:

"Please create a JSON file for a language learning app. The topic is [Your Topic, e.g., Business English]. It should have a 'title', a 'default_type' of 'sentence', and a list of 'translations'. Each item needs 'source' (English), 'target' (Korean), and a 'context' tag. Please include 10 useful phrases."

3. How to Import to App

Once you have your .json file:

  1. Email to Self: Send the file to your own email address. Open the email on your phone and download the file.
  2. USB Transfer: Connect your phone to your PC via USB and copy the file to your "Downloads" folder.
  3. In App: Go to Mode 2 (Review), tap the Import Icon (‚¨áÔ∏è), and select your file.
