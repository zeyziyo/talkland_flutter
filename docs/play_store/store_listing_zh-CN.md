# Title
Talkie: AI 语言导师 (口语 & 翻译)

# Short Description (80 chars)
AI 助你掌握外语。实时口语练习、即时翻译以及自动复习功能。

# Full Description
Talkie 是专为口语练习设计的智能语言学习伙伴，助你唤醒沉睡的语言潜能。

**主要功能:**

🎙️ **实时口语练习:**
Talkie 会倾听你的发音，为你提供一个可以安心练习口语的空间。使用“口语模式 (Speaking Mode)”，设置 3 到 60 秒的间隔，按照自己的节奏进行跟读 (Shadowing) 和重复训练。

🔍 **智能搜索 & 翻译:**
支持韩语、英语、日语、中文等多种语言的即时文本翻译。Talkie 会自动检测你之前学习过的相似句子，防止重复学习。

📖 **系统化复习:**
学过的内容不再遗忘。保存翻译内容和学习资料，通过系统化的复习机制，将其转化为长期记忆。

📄 **自定义学习资料:**
通过 JSON 文件导入你专属的学习列表。根据教科书或特定兴趣领域定制你的学习体验。

**🌍 多语言支持 (43 种语言):**
Talkie 支持 43 种语言的翻译和学习:
- **东亚:** 韩语, 日语, 中文 (简体/繁体)
- **欧洲:** 英语, 西班牙语, 法语, 德语, 意大利语, 葡萄牙语, 俄语 等
- **南亚:** 印地语, 孟加拉语, 泰米尔语 等
- **东南亚:** 印尼语, 越南语, 泰语, 菲律宾语 等
- **中东及非洲:** 阿拉伯语, 土耳其语, 斯瓦希里语 等

**灵活的套餐计划 (Flexible Plans):**
- **基础版 (Free):** 每天免费提供 5 次翻译。观看一个广告即可**立即补充 5 次**，让你免费持续学习。
- **专业版 (Pro):** 升级去除广告，每月 1,500 次翻译（平均每天 50 次），让你专注于学习不受打扰。

## 自定义学习资料示例 (JSON)
创建你自己的单词本 `my_study.json` 并导入：
```json
{
  "source_language": "en",
  "target_language": "zh-CN",
  "subject": "旅行会话",
  "entries": [
    {"source_text": "Where is the station?", "target_text": "车站在哪里？"},
    {"source_text": "How much is this?", "target_text": "这个多少钱？"}
  ]
}
```

**为什么选择 Talkie?**
- **隐私优先:** 语音数据安全处理，不会永久存储。
- **离线优先:** 学习记录仅保存在你的设备上。
- **简洁 UI:** 专注于学习的无干扰界面。
