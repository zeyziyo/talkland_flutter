# Title
Talkie: AI 언어 튜터 (회화 & 번역)

# Short Description (80 chars)
AI와 함께하는 실전 회화 연습. 즉시 번역, 말하기 훈련, 그리고 자동 복습까지.

# Full Description
말하기 연습을 위해 설계된 지능형 AI 언어 학습 파트너, Talkie와 함께 잠들어 있던 언어 잠재력을 깨워보세요.

**주요 기능:**

🎙️ **실시간 회화 연습:**
Talkie는 당신의 발음을 듣고 안전하게 말하기 연습을 할 수 있는 공간을 제공합니다. "Speaking Mode"를 사용하여 3초에서 60초까지 간격을 설정하고, 나만의 속도로 쉐도잉(Shadowing)과 반복 훈련을 해보세요.

🔍 **똑똑한 검색 & 번역:**
한국어, 영어, 일본어, 중국어 등 다양한 언어로 텍스트를 즉시 번역하세요. Talkie는 이전에 학습한 비슷한 문장을 자동으로 감지하여 중복을 방지해 줍니다.

📖 **체계적인 복습:**
한 번 배우고 잊어버리지 마세요. 번역 내용과 학습 자료를 저장하고, 체계적인 복습 시스템을 통해 장기 기억으로 만드세요.

📄 **나만의 학습 자료:**
JSON 파일을 통해 나만의 학습 리스트를 가져오세요. 교과서나 특정 관심 분야에 맞춰 학습 경험을 커스터마이징할 수 있습니다.

**🌍 다국어 지원 (43개 언어):**
Talkie는 43개 언어에 대한 번역과 학습을 지원합니다:
- **동아시아:** 한국어, 일본어, 중국어 (간체/번체)
- **유럽:** 영어, 스페인어, 프랑스어, 독일어, 이탈리아어, 포르투갈어, 러시아어 등
- **남아시아:** 힌디어, 벵골어, 타밀어 등
- **동남아시아:** 인도네시아어, 베트남어, 태국어, 필리핀어 등
- **중동 및 아프리카:** 아랍어, 튀르키예어, 스와힐리어 등

**합리적인 요금제 (Flexible Plans):**
- **기본 (Free):** 매일 5회 무료 번역으로 가볍게 시작하세요. 광고 하나를 보면 **즉시 5회 충전**되어 계속 무료로 공부할 수 있습니다.
- **프로 (Pro):** 업그레이드하여 광고를 제거하고, 월 1,500회 (일 평균 50회) 넉넉한 번역으로 끊김 없이 몰입하세요.

## 나만의 학습 자료 예시 (JSON)
나만의 단어장을 `my_study.json` 파일로 만들어 가져와보세요:
```json
{
  "source_language": "en",
  "target_language": "ko",
  "subject": "여행 회화",
  "entries": [
    {"source_text": "Where is the station?", "target_text": "역이 어디에 있나요?"},
    {"source_text": "How much is this?", "target_text": "이거 얼마예요?"}
  ]
}
```

**왜 Talkie인가요?**
- **개인정보 보호:** 음성 데이터는 안전하게 처리되며 영구 저장되지 않습니다.
- **오프라인 우선:** 학습 기록은 당신의 기기에만 저장됩니다.
- **깔끔한 UI:** 오직 학습에만 집중할 수 있는 심플한 인터페이스.

지금 바로 자신 있게 말하기 시작하세요!
