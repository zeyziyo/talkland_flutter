# Title
Talkie: AI Language Tutor (会話 & 翻訳)

# Short Description (80 chars)
AIと一緒に実践会話練習。即時翻訳、スピーキング訓練、そして自動復習まで。

# Full Description
スピーキング練習のために設計されたインテリジェントAI語学パートナー、Talkieで眠っていた語学の潜在能力を目覚めさせましょう。

**主な機能:**

🎙️ **リアルタイム会話練習:**
Talkieはあなたの発音を聞き、安全にスピーキング練習ができる空間を提供します。「Speaking Mode」を使い、3秒から60秒まで間隔を設定して、自分だけのペースでシャドーイング(Shadowing)と反復訓練を行いましょう。

🔍 **スマート検索 & 翻訳:**
韓国語、英語、日本語、中国語など多彩な言語へテキストを即座に翻訳します。Talkieは以前学習した類似の文章を自動的に検出し、重複を防ぎます。

📖 **体系的な復習:**
一度学んだことを忘れないでください。翻訳内容と学習資料を保存し、体系的な復習システムを通じて長期記憶として定着させましょう。

📄 **自分だけの学習資料:**
JSONファイルを通じて自分だけの学習リストを取り込みましょう。教科書や特定の関心分野に合わせて学習体験をカスタマイズできます。

**🌍 多言語サポート (43言語):**
Talkieは43言語の翻訳と学習をサポートします:
- **東アジア:** 韓国語、日本語、中国語 (簡体/繁体)
- **ヨーロッパ:** 英語、スペイン語、フランス語、ドイツ語、イタリア語、ポルトガル語、ロシア語など
- **南アジア:** ヒンディー語、ベンガル語、タミル語など
- **東南アジア:** インドネシア語、ベトナム語、タイ語、フィリピン語など
- **中東およびアフリカ:** アラビア語、トルコ語、スワヒリ語など

**合理的な料金プラン (Flexible Plans):**
- **基本 (Free):** 毎日5回の無料翻訳で気軽に始めましょう。広告を1つ見ると**即座に5回分チャージ**され、無料で学習を続けられます。
- **プロ (Pro):** アップグレードして広告を削除し、月1,500回 (1日平均50回) の余裕ある翻訳で、途切れることなく学習に没頭しましょう。

## カスタム学習資料の例 (JSON)
自分だけの単語帳を `my_study.json` ファイルとして作成し、取り込んでみてください:
```json
{
  "source_language": "en",
  "target_language": "ja",
  "subject": "旅行会話",
  "entries": [
    {"source_text": "Where is the station?", "target_text": "駅はどこですか？"},
    {"source_text": "How much is this?", "target_text": "これはいくらですか？"}
  ]
}
```

**なぜTalkieなのか?**
- **プライバシー重視:** 音声データは安全に処理され、永久に保存されることはありません。
- **オフライン優先:** 学習記録はあなたのデバイスにのみ保存されます。
- **シンプルなUI:** 学習のみに集中できるシンプルなインターフェース。

自信を持って話し始めましょう！
