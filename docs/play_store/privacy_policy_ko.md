# Talkie 개인정보처리방침 (Privacy Policy)

**최종 수정일:** 2026-01-09

Talkie(이하 "회사" 또는 "앱")는 사용자의 개인정보를 소중하게 보호하며, 본 방침을 통해 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

## 1. 수집하는 정보

### 마이크 데이터 (음성)
본 앱은 음성 인식 기능을 수행하기 위해 기기의 마이크 권한(`android.permission.RECORD_AUDIO`)을 필요로 합니다.
- **사용 목적**: 마이크 아이콘을 눌렀을 때 사용자의 음성을 텍스트로 변환하기 위해 사용됩니다.
- **저장 여부**: 사용자의 음성 데이터는 **영구 저장되지 않습니다**. 해당 데이터는 실시간 처리를 위해서만 사용되거나 음성 인식 서비스(STT)로 전송되어 번역 목적으로만 사용된 후 즉시 파기됩니다.

### 학습 데이터
소스 텍스트, 번역된 텍스트, 복습 기록 등의 학습 데이터는 사용자 기기 내부의 데이터베이스에 로컬로 안전하게 저장됩니다. 이 데이터는 저희 서버로 전송되지 않습니다.

## 2. 제3자 서비스 (Third-Party Services)
본 앱은 음성 인식, 번역 및 광고 기능을 위해 다음과 같은 제3자 서비스를 사용할 수 있으며, 이들 제공자는 자체적인 개인정보처리방침에 따라 데이터를 처리할 수 있습니다.

### 광고 (Google AdMob)
이 앱은 Google AdMob을 사용하여 광고를 표시합니다. AdMob은 다음과 같은 정보를 사용할 수 있습니다:
- **광고 ID (Advertising ID)**: 맞춤형 광고 제공을 위한 기기 식별자.
- **쿠키 (Cookies)**: 광고 상호작용 정보 수집 목적.
기기 설정에서 광고 추적 제한(예: "개인 맞춤광고 선택 해제")을 설정하여 프라이버시를 관리하실 수 있습니다.
- [Google 광고 정책 보기](https://policies.google.com/technologies/ads)

### 기능 제공 서비스
- **Google ML Kit / Speech Services**: 음성 인식 기능 수행.
- **Google Cloud Translation API**: 텍스트 번역 기능 수행.

## 3. 데이터 보안
사용자의 모든 학습 데이터는 기기에 로컬로 저장되므로, 기기 자체의 보안 수준에 따릅니다. 기기의 안드로이드 버전을 최신으로 유지하시기를 권장합니다.

## 4. 아동 개인정보 보호
본 앱은 만 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 5. 정책 변경
법률이나 서비스의 변경 사항을 반영하기 위해 본 개인정보처리방침을 수정할 수 있습니다. 중요한 변경이 있을 경우 앱 내 공지사항을 통해 안내해 드립니다.
