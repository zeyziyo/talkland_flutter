# Title
Talkie: AI 語言導師 (口說 & 翻譯)

# Short Description (80 chars)
AI 助你掌握外語。即時口說練習、翻譯以及自動複習功能。

# Full Description
Talkie 是專為口說練習設計的智能語言學習夥伴，助你喚醒沉睡的語言潛能。

**主要功能:**

🎙️ **即時口說練習:**
Talkie 會傾聽你的發音，為你提供一個可以安心練習口說的空間。使用「口說模式 (Speaking Mode)」，設置 3 到 60 秒的間隔，按照自己的節奏進行跟讀 (Shadowing) 和重複訓練。

🔍 **智能搜尋 & 翻譯:**
支持韓語、英語、日語、中文等多種語言的即時文本翻譯。Talkie 會自動檢測你之前學習過的相似句子，防止重複學習。

📖 **系統化複習:**
學過的內容不再遺忘。保存翻譯內容和學習資料，通過系統化的複習機制，將其轉化為長期記憶。

📄 **自定義學習資料:**
通過 JSON 文件導入你專屬的學習列表。根據教科書或特定興趣領域定制你的學習體驗。

**🌍 多語言支持 (43 種語言):**
Talkie 支持 43 種語言的翻譯和學習:
- **東亞:**韓語, 日語, 中文 (簡體/繁體)
- **歐洲:** 英語, 西班牙語, 法語, 德語, 意大利語, 葡萄牙語, 俄語 等
- **南亞:** 印地語, 孟加拉語, 泰米爾語 等
- **東南亞:** 印尼語, 越南語, 泰語, 菲律賓語 等
- **中東及非洲:** 阿拉伯語, 土耳其語, 斯瓦希里語 等

**靈活的方案 (Flexible Plans):**
- **基礎版 (Free):** 每天免費提供 5 次翻譯。觀看一個廣告即可**立即補充 5 次**，讓你免費持續學習。
- **專業版 (Pro):** 升級去除廣告，每月 1,500 次翻譯（平均每天 50 次），讓你專注於學習不受打擾。

## 自定義學習資料範例 (JSON)
創建你自己的單字本 `my_study.json` 並導入：
```json
{
  "source_language": "en",
  "target_language": "zh-TW",
  "subject": "旅行會話",
  "entries": [
    {"source_text": "Where is the station?", "target_text": "車站在哪裡？"},
    {"source_text": "How much is this?", "target_text": "這個多少錢？"}
  ]
}
```

**為什麼選擇 Talkie?**
- **隱私優先:** 語音數據安全處理，不會永久存儲。
- **離線優先:** 學習記錄僅保存在你的設備上。
- **簡潔 UI:** 專注於學習的無干擾介面。
