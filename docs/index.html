Talkie 사용자 가이드

언제 어디서나 완벽하게 외국어를 정복하세요.

4가지 핵심 학습 모드

1. 입력 (음성/번역)

마이크를 눌러 음성으로 입력하거나 직접 텍스트를 입력하세요. 80개국 실시간 번역을 지원하며 즉시 발음을 들어볼 수 있습니다. '데이터 저장'을 누르면 복습 장부로 넘어갑니다.

2. 복습 (플래시카드)

저장하거나 다운로드한 문장들을 암기합니다. '보이기/숨기기'로 뜻을 가릴 수 있고, 완전히 암기한 카드는 체크(✓)를 눌러 숨길 수 있습니다.

3. 발음 연습 (쉐도잉)

가장 강력한 스피킹 모드입니다. 원어민 음성을 듣고, 간격(-, +초) 안에 동일하게 따라 말해보세요. 음성 인식을 통해 정확도(0~100점)를 즉시 평가받습니다.

4. AI 실전 채팅

다양한 페르소나(영어 선생님, 길안내원 등)를 설정하고 실전 대화를 나눕니다. GPS 기반으로 회원님의 현재 동네 주변 명소를 추천받는 현장감 넘치는 대화도 가능합니다.

나만의 JSON 파일 가져오기 (대량 학습)

Talkie의 진짜 매력은 내가 정리한 문장들을 한 번에 넣어서 훈련하는 것에 있습니다. 메모장에 아래 양식대로 적은 후 .json 파일로 저장하여 앱 바 메뉴의 [기기에서 가져오기]를 선택하세요.

📌 문장(Sentence) 넣기 템플릿

{
  "subject": "공항 출입국 심사 (Airport)",
  "entries": [
    {
      "source_text": "여권 좀 보여주시겠습니까?",
      "translated_text": "May I see your passport, please?",
      "tags": ["공항", "입국심사", "필수"]
    },
    {
      "source_text": "며칠 동안 머무르실 예정인가요?",
      "translated_text": "How long will you be staying?",
      "tags": ["공항", "입국심사"]
    }
  ]
}

📌 AI 실전 대화(Dialogue) 넣기 템플릿

대화 기록 탭에서 불러와 읽어볼 수 있으며, 발음이나 역할별 번역 등을 한눈에 볼 수 있습니다.

{
  "default_type": "dialogue",
  "subject": "호텔 체크인 상황극",
  "participants": [
    {
      "id": "p1",
      "name": "Jordan",
      "role": "User",
      "gender": "Male",
      "lang_code": "en-US"
    },
    {
      "id": "p2",
      "name": "프론트 직원",
      "role": "AI",
      "gender": "Female",
      "lang_code": "ko-KR"
    }
  ],
  "entries": [
    {
      "text": "Hello! I have a reservation under the name Jordan.",
      "speaker": "Jordan"
    },
    {
      "text": "안녕하세요, 조단 님. 예약이 확인되었습니다.",
      "speaker": "프론트 직원"
    }
  ]
}

Tip: ChatGPT나 Claude 같은 AI에게 "위 JSON 포맷에 맞춰서 회사 비즈니스 미팅에서 쓸 10가지 문장을 만들어줘!" 라고 입력하면 매우 빠르고 편하게 자료를 생성할 수 있습니다.
